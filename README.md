### SENTINEL's Solution

```

вң… Notify: Sends confirmation to user
```

### The Tech Behind It

**Dual Detection System:**

#### 1. **Balance Monitoring** (Fast)
```javascript
// Every 60 seconds
const currentBalance = await treasury.getBalance();

  // Someone sent us money!
  await matchTransferToLoans(increase);

```

#### 2. **WDK Indexer API** (Robust)
```javascript
// Query real transaction history
const transfers = await indexer.getTokenTransfers(treasuryAddress, {
  chain: 'ethereum-sepolia',
  token: 'usdt',
  startDate: loan.disbursedAt
});

// Find matching repayment
const repayment = transfers.find(tx =>
  tx.from === borrower.wallet &&
  Math.abs(tx.amount - loan.totalDue) < 0.01
);

// Verify transaction on-chain
const verified = await indexer.verifyTransaction(tx.hash, {
  from: borrower.wallet,
  to: treasuryAddress,
  minAmount: loan.totalDue
});

// Auto-mark as repaid
if (verified) {
  await loan.markRepaid(tx.hash);
  await credit.updateScore(+5); // Reward on-time repayment
}
```
