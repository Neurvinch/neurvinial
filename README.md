## 🔍 Autonomous On-Chain Detection (No Human Needed!)

**This is where it gets INSANE.**

### The Problem with Other Bots

```
❌ User: /repay
❌ Bot: "Ok marked as repaid"
❌ Reality: Bot trusts user, no verification
❌ Result: Users can lie, no blockchain proof
```

### Neurvinial's Solution

```
✅ User: Sends USDT to treasury (on Ethereum)
✅ Monitor: Detects balance increase (polls every 60s)
✅ Indexer: Queries WDK Indexer API for TX details
✅ Matcher: Matches TX amount to outstanding loan
✅ Auto-Update: Marks loan repaid + updates credit score
✅ Notify: Sends confirmation to user
✅ Result: 100% verifiable, zero trust needed!
```

### The Tech Behind It

**Dual Detection System:**

#### 1. **Balance Monitoring** (Fast)
```javascript
// Every 60 seconds
const currentBalance = await treasury.getBalance();
if (currentBalance > lastBalance) {
  // Someone sent us money!
  await matchTransferToLoans(increase);
}
```

#### 2. **WDK Indexer API** (Robust)
```javascript
// Query real transaction history
const transfers = await indexer.getTokenTransfers(treasuryAddress, {
  chain: 'ethereum-sepolia',
  token: 'usdt',
  startDate: loan.disbursedAt
});

// Find matching repayment
const repayment = transfers.find(tx =>
  tx.from === borrower.wallet &&
  Math.abs(tx.amount - loan.totalDue) < 0.01
);

// Verify transaction on-chain
const verified = await indexer.verifyTransaction(tx.hash, {
  from: borrower.wallet,
  to: treasuryAddress,
  minAmount: loan.totalDue
});

// Auto-mark as repaid
if (verified) {
  await loan.markRepaid(tx.hash);
  await credit.updateScore(+5); // Reward on-time repayment
}
```
